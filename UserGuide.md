# 功能

## `B站`直播订阅

-   [x] 管理员权限

> 会在订阅的人开播、下播、轮播时提醒

1. `直播订阅 [用户uid] [昵称]`
2. `取消直播订阅 [用户uid]`
3. `取消直播订阅 [昵称]`

## `B站`视频订阅

-   [x] 管理员权限
    > 会在订阅的人投稿视频时提醒

1. `视频订阅 [用户uid] [昵称]`
2. `取消视频订阅 [用户uid]`
3. `取消视频订阅 [昵称]`

## `B站`动态订阅

-   [x] 管理员权限
    > 会在订阅的人发任何动态时提醒

1. `动态订阅 [用户uid] [昵称]`
2. `取消动态订阅 [用户uid]`
3. `取消动态订阅 [昵称]`
